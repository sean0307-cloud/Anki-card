- patient → **im**patient（有耐心 → 沒耐心）
- perfect → **im**perfect（完美 → 不完美）
- polite → **im**polite（有禮 → 失禮）
- possible → **im**possible（可能 → 不可能）
- practical → **im**practical（實際 → 不實際）
- dependent → **in**dependent（依賴 → 獨立）
- regular → **ir**regular（規律 → 不規律）
- responsible → **ir**responsible（負責 → 不負責）
